match.names
===========

描述
    在数据中将名字与一个部分名称匹配列表进行匹配。通常，一个信号或指标的通用定义将包括部分名称匹配。在金融数据中，
    常用的部分匹配包括“close”、“open”和“volume“，但还有很多。

用法
::

    match.names(match_names, data_names)

参数
    :data_names: 数据匹配的名字
    :match_names: 要匹配的名字

说明
    在复杂数据中，补充名字信息将列名字。例如，添加上证券代码或一些调整的指标。
    这个小实用工具的存在是为了在中心位置进行匹配以便能执行更健壮的错误处理和报告。
